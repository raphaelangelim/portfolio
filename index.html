The Injected Ad Component

Zero CLS Strategy

The Challenge: Injecting dynamic third-party ads into article streams without causing frustrating content jumps (Layout Shifts) or slowing down the Main Thread.

Visual Solution

[ACTION: DROP IMAGE HERE]

Recommended: A GIF or screenshot of the
skeleton placeholder in the layout.

Result: Created a "slot-based" system that reserves exact vertical space before the ad loads, ensuring a smooth reading experience.

Code Implementation

/* Reserve space to prevent CLS */
.ad-slot {
  min-height: 250px;
  background: var(--color-bg-offset);
  display: grid;
  place-items: center;
  content-visibility: auto;
  contain-intrinsic-size: 250px;
}

/* Skeleton placeholder UI */
.ad-slot:empty::after {
  content: 'ADVERTISEMENT';
  font-size: 10px;
  opacity: 0.5;
}

Approach: Utilized min-height and content-visibility to optimize browser rendering and prevent re-paints.

Conversion Focused Case Study

Interactive Multi-Video Campaign Slider

HTML CSS UX Logic

Objective was to promote a Winter Camp program through an interactive video component embedded in high-traffic partner pages while keeping performance and UX intact.

Visual Implementation

The component uses auto-playing muted videos with a clear call to action, effectively reducing friction between interest and lead generation via WhatsApp.

Core Logic Example 

function playVideo() {
    videos.forEach(v => {
        v.pause();
        v.currentTime = 0;
    });

    const activeVideo = videos[currentIndex];
    activeVideo.play().catch(() => {});

    activeVideo.onended = () => {
        setTimeout(() => move(1), 400);
    };
}

The video lifecycle directly controls the slider behavior, ensuring a smooth, automated flow that respects user engagement.

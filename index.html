HTML CSS UX Logic

Objective was to promote a Winter Camp program through an interactive video component embedded in high traffic partner pages while keeping performance and UX intact.

Visual Implementation

The component uses auto playing muted videos with clear call to action, reducing friction between interest and lead generation.

Core Logic Example 

function playVideo() {
    videos.forEach(v => {
        v.pause();
        v.currentTime = 0;
    });

    const activeVideo = videos[currentIndex];
    activeVideo.play().catch(() => {});
